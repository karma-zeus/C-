📋 Project Overview

This is a core FPS (First-Person Shooter) game engine implementation in C++ using OpenGL. It features a complete game loop, entity system, physics simulation, and combat mechanics.

⭐ Key Features

🎨 Graphics Engine

OpenGL-based rendering with shader support for stunning visuals

⚙️ Physics System

Rigid body physics with gravity simulation for realistic gameplay

🎮 Input System

Keyboard and mouse handling with smooth camera control

🕹️ Entity System

Player, Enemy, and Projectile entities for dynamic gameplay

🌍 World System

Level generation and object management

🔊 Audio Framework

Ready for complete sound integration

🎯 Gameplay Mechanics

  • First-Person Camera: Mouse look and WASD movement
  • Combat System: Projectile firing and enemy AI
  • Damage System: Health tracking for player and enemies
  • Enemy AI: Chase and attack behavior
  • HUD System: Health, ammo, and kill counter display

📦 Requirements

  • C++ 17 or later
  • CMake 3.10 or later
  • GLFW3
  • OpenGL 3.3+
  • GLM (header-only)

🚀 Quick Start

Installation (Ubuntu/Debian)

sudo apt-get install libglfw3-dev libglm-dev libgl1-mesa-dev

Building

mkdir build cd build cmake .. make ./fps_game

Controls

  • WASD: Movement
  • Mouse: Look around
  • Left Click: Shoot
  • ESC: Exit
