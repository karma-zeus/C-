🎮 Game Controls

⌨️ WASD
Move forward, backward, left, right
🖱️ Mouse
Look around and aim
🔫 Left Click
Shoot projectiles at enemies
⎋ ESC
Exit or pause game
🎯 Space
Jump (if enabled)
🏃 Shift
Sprint (if enabled)

🎨 Modern Graphics

OpenGL-based rendering with shader support for stunning visuals and smooth gameplay.

⚙️ Physics Engine

Realistic physics simulation with gravity, collisions, and projectile dynamics.

🕹️ Entity System

Fully featured entity system with player, enemy AI, and interactive projectiles.

🌍 Dynamic World

Procedurally generated levels with dynamic object management and environmental interactions.

🤖 Enemy AI

Intelligent enemies with chase and attack behavior for challenging gameplay.

📊 HUD System

Complete heads-up display showing health, ammo, kills, and game statistics.
